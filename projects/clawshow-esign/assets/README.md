# Assets — ClawShow eSign MVP

**This MVP uses ZERO external assets.** No images, video clips, fonts, audio, or stock
footage are required. Every scene is rendered programmatically from existing OpenMontage /
Remotion React components (`StatCard`, `SectionTitle`) over an animated CSS background.

That is what keeps it **zero-key, no-GPU, no paid API** — and fully reproducible: anyone who
clones the repo can render the exact same video with no downloads beyond the npm packages.

## Why empty

- Headlines, subtitles, and the CTA are **text props** in
  [`clawshow-esign.json`](../../../remotion-composer/public/demo-props/clawshow-esign.json).
- Colors come from the `themeConfig` block in that same JSON.
- The background motion is generated by the composition, not by media files.

## If you want to add assets later (optional)

Drop files in this folder and reference them from the JSON. Relevant `cut` fields the
`Explainer` composition already supports:

- `backgroundImage`: a still rendered behind a card (e.g. a product screenshot). Use an
  absolute path or a path relative to `remotion-composer/public/`.
- `backgroundVideo`: a short loop behind a card (takes priority over `backgroundImage`).
- `source` + `type` omitted: a full-bleed image/video scene (Ken Burns / playback).
- `audio.narration` / `audio.music`: voiceover or background track (the zero-key path uses
  Piper TTS for narration — still no paid API).

Keep additions optional: the video must continue to render with this folder empty.
